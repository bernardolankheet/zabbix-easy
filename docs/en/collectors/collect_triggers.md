# `CollectTriggers` / `CollectTriggersCount`

Collectors for trigger listing and counts. Useful for KPIs and trigger-table generation.

Usage

- `CollectTriggers(apiUrl, token, state, req)` — returns triggers filtered by state
- `CollectTriggersCount(apiUrl, token, req)` — returns numeric count
