# `CollectHosts` / `CollectHostExists`

Hosts listing and existence checks. Returns host objects or boolean existence.

Usage

- `CollectHosts(apiUrl, token, req)` — returns list of hosts and metadata
- `CollectHostExists(apiUrl, token, hostid, req)` — returns `(bool, error)`
