# `CollectProxiesList`

Retorna uma lista leve de proxies (IDs e contagens) adequada para resumos e consultas rápidas.

Uso

- Assinatura: `CollectProxiesList(apiUrl, token string, req ApiRequester) ([]map[string]interface{}, error)`
- Chamadas: `proxy.get` com `output` mínimo
